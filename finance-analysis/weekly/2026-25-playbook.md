## 悲观情景

| 项目 | 内容 |
| --- | --- |
| 触发条件 | 议息后指数放量破位，科创50 高位长阴，科技材料和算力金属同步补跌，金融或防守方向无法托住指数 |
| 风险信号 | 成交额放大但承接失败，强势 ETF 集体破 10 日或 20 日线，PCB 上游和半导体材料反包失败 |
| 处理原则 | 降低新开仓优先级，保留现金和复盘空间；先观察失败样本，不急着把回调解释成机会 |
| 需要降权的假设 | H-20260610-001、H-20260611-001、H-20260614-001、H-20260617-001 |
| 防错重点 | 不把“长期方向好”直接转成短期扛单；不把 ETF 分散误解成没有回撤 |

## 关键指标

| 指标 | 看什么 | 说明错了的情况 |
| --- | --- | --- |
| 科创50 和创业板 | 是否继续强于沪深300和上证50 | 科创50 高位放量回落，创业板无法维持修复 |
| 成交额 | 高成交是否带来承接和扩散 | 高成交只带来高波动，强势方向轮番冲高回落 |
| 科技材料第二波 | 前低、五日线、趋势线、反包力度 | 反包后继续放量破位 |
| 半导体设备材料 | 设备、材料、对日替代是否强于泛芯片 | 科创芯片宽基持续强于科创半设，设备材料失去相对收益 |
| ETF 成分质量 | 指数成分是否对应业绩线和高端制造 | 主题相近但成分业绩弱，切换后跑输原指数 |
| 涨价链 | 钨、钼、铜、PCB 上游、电子布电子纱是否有价格或订单确认 | 只有题材扩散，没有价格和业绩证据 |

## 触发条件

| 条件 | 进入观察 | 降低优先级 |
| --- | --- | --- |
| 半导体设备材料 | 科创半设强于科创芯片，设备材料分支放量但不滞涨 | 科创芯片宽基更强，设备材料高开低走 |
| PCB 上游 | 电子布、铜箔、树脂、覆铜板继续有价格和订单线索 | 上游涨价不能传导，中游和下游同步回落 |
| 5G 和通信 | 5G 指数强于通信设备指数，互联网通信和 6G 叙事有承接 | 通信设备指数反而更强，5G 指数没有相对收益 |
| 消费电子高端制造 | 立讯、京东方、东山等高端制造属性强于普通消费叙事 | 纯消费电子逻辑无法兑现业绩 |
| 高股息和水电 | 来水、发电量、利润改善与低波动共振 | 利率上行或科技风险偏好回升导致持续跑输 |

## 不行动的理由

| 情况 | 为什么不急 |
| --- | --- |
| 只有主题名，没有成分拆解 | ETF 和板块都可能同名不同质，先看成分和业绩 |
| 只有单日大涨，没有回调确认 | 高位追入容易遇到议息后兑现 |
| 只有宏观乐观，没有分支承接 | 美联储不恶化不等于所有成长方向都能涨 |
| 只有长期逻辑，没有短期失效条件 | 国产替代和涨价都要写清订单、毛利率、价格和破位条件 |
| 已经明显赚钱但事件未落地 | 可以先保护利润垫，再等事件后重新评估 |
